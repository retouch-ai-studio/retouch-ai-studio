## 🤖 Как работает автоматическое обновление (GitHub Actions)

В репозиторий встроен робот автообновления (`.github/workflows/update_site.yml`).
Каждую ночь в 03:00 по московскому времени робот:
1. Запускает скрипт `scripts_and_raw_data/fetch_all_vk.py` для проверки новых постов в группе ВК.
2. При обнаружении новых кейсов обновляет базу данных `vk_posts_all.json`.
3. Запускает `scripts_and_raw_data/build_site.py` для пересборки файла `index.html`.
4. Автоматически публикует свежую версию сайта на **GitHub Pages**.
